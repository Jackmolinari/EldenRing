# EldenRing

Development of a smart ring based on the ESP32-C3 microcontroller.

## Project objectives

- Heart-rate monitoring
- SpO2 measurement
- Bluetooth Low Energy communication
- Smartphone data visualization
- Rechargeable battery
- Custom PCB
- Wearable enclosure

## Current hardware

- ESP32-C3 SuperMini
- MAX30102 sensor
- LCD1602 with I2C backpack
- Breadboard development setup
